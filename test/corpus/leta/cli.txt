leta --help
---
Usage: leta [OPTIONS] COMMAND [ARGS]...

  Fast semantic code search powered by Language Server Protocol.

  Find definitions, references, and navigate code using LSP servers. Works
  with any language that has an LSP implementation.

Options:
  --json      Output in JSON format
  --version   Show the version and exit.
  -h, --help  Show this message and exit.

Commands:
  calls            Show call hierarchy for a symbol.
  config           Show configuration info and paths.
  daemon           Manage the leta daemon.
  declaration      Find declaration(s) of a symbol.
  diagnostics      Show diagnostics (errors, warnings) for files.
  edit             Edit files in the workspace.
  files            List workspace files with symbol counts.
  grep             Search for symbols matching a regex pattern.
  help-all         Show detailed help for all commands.
  hover            Show hover information for a symbol.
  implementations  Find implementations of an interface or abstract method.
  mv               Move/rename a file and update imports.
  ref              Alias for 'refs'.
  refs             Find all references to a symbol.
  rename           Rename a symbol across the workspace.
  show             Show definition(s) of a symbol with full source code.
  subtypes         Find subtypes (implementing classes) of a type.
  supertypes       Find supertypes (base classes/interfaces) of a type.
  workspace        Manage workspaces.

leta show --help
---
Usage: leta show [OPTIONS] SYMBOL

  Show definition(s) of a symbol with full source code.

  SYMBOL can be:   - SymbolName: search by name   - Parent.Symbol: qualified
  name (e.g., Class.method)   - path:Symbol: filter by file path (supports
  globs)   - path:Parent.Symbol: combine path and qualified name

  Examples:     leta show User     leta show UserRepository.save     leta show
  "models/*:User"     leta show --head 50 LARGE_CONSTANT

Options:
  -n, --context-lines INTEGER  Number of context lines before/after
  --head INTEGER               Limit output to first N lines
  -h, --help                   Show this message and exit.

help grep

leta grep --help
---
Usage: leta grep [OPTIONS] PATTERN [PATH]

  Search for symbols matching a regex pattern.

  Only searches symbol NAMES, not file contents. Use ripgrep for full-text
  search.

  Examples:     leta grep "^test_"                    # Functions starting
  with test_     leta grep "Handler$" -k class        # Classes ending with
  Handler     leta grep ".*" src/models.py -k class  # All classes in a file
  leta grep "parse" -d                  # Show docstrings

Options:
  -k, --kind TEXT              Filter by symbol kind (e.g., class, function,
                               method)
  -d, --docs                   Include documentation/docstrings
  -x, --exclude TEXT           Exclude files/directories matching pattern
                               (repeatable)
  -C, --case-sensitive         Case-sensitive matching (default: case-
                               insensitive)
  --head INTEGER               Limit output to first N lines
  -i, --include-non-workspace  Include symbols from external dependencies
  -h, --help                   Show this message and exit.
